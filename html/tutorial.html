Tutorial

A presente proposta tem como intuito desenvolver um projeto web, com finalidade de ensinar lógica de programação adaptado para o público infantil para que possa facilitar e auxiliar o aprendizado e raciocínio lógico dos mesmos. O projeto apresentado servirá como um auxílio na educação de novos estudantes. Portanto, espera-se que o uso do projeto proporcione uma maior facilidade na trajetória dos usuários que pretendam seguir na área de TI ou afins, tendo uma interface divertida e de fácil entendimento e usabilidade.

O objetivo geral do projeto web será incentivar e adaptar público leigo com a lógica de programação. A maior preocupação é com ensino do usuário em relação ao assunto proposto, mas nada que deixe a experiência em acessar entediante, por isso outra grande prioridade é a relação atrativa que o jogo vai estabelecer com o acessante. E a relação entre o assunto e o usuário tem como objetivo passar o máximo de clareza possível, para não deixar o usuário desorientado e com entendimento do conteúdo.

A lógica de programação não é algo fácil de se compreender, exige conhecimento e prática do conteúdo estudado. Mesmo tendo alguma experiência no ramo, aprender uma nova linguagem de programação nunca é simples. Agora imagine, uma pessoa entrando neste “mundo” de muitas informações. Complicado não é? Algo que colabore para o saber e o incentivo á prática necessária para alcançar o êxito esperado no assunto seria aceito naturalmente e ideal para circunstância.
