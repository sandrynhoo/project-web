Tutorial

A presente proposta tem como intuito desenvolver um projeto web, com finalidade de ensinar lógica de programação adaptado para o público infantil para que possa facilitar e auxiliar o aprendizado e raciocínio lógico dos mesmos. O projeto apresentado servirá como um auxílio na educação de novos estudantes. Portanto, espera-se que o uso do projeto proporcione uma maior facilidade na trajetória dos usuários que pretendam seguir na área de TI ou afins, tendo uma interface divertida e de fácil entendimento e usabilidade.
